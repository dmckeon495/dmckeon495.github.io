ABOUT DAN

Coming from a background in literary, journalistic, and legal editing, Dan McKeon is an LGBTQ coder-in-training in both front-end and back-end web development. Dan graduated from the University at Buffalo in 2018, spending three years working for The Spectrum, the student-run newspaper, as a reporter, copy chief and managing editor, as well as three years as an editor for NAME Magazine, the literary magazine.

Out of college, Dan worked for General Code editing municipal ordinances into code books available online in Rochester, NY. They currently work in Philadelphia, PA as a shift manager at Starbucks, balancing coding projects and brewing coffee.

Dan also enjoys reading and writing poetry, playing and modding video games, and programming a Twitter bot to dunk on them.

PLOND HOCKEY

Plond Hockey is a browser-based hockey simulation game. Teams such as the Burlington Lumberjacks and Philly Pineapples play 3v3 hockey games every hour, run on a JavaScript sim I made from scratch. The game also tracks stats and league rankings. It's still a work-in-progress I plug away it in my free time. Plond Hockey uses a MERN stack.

GitHub link: Git Repository
Application link: Plond Hockey

DOOSKBOT

Dooskbot is a Twitter bot that runs on Node.js made as a side project to try out concepts learned in class. It uses a bank of nouns, verbs, adjectives and other parts of speech to create tweets based on a collection of templates. The bot uses RNG to pick both the type of tweet and the specific words it will use in the tweet, as well as replies, and then automatically sends it out to the World.

GitHub link: Git Repository
Twitter link: @dooskbot

UPTRADE

UpTrade is a video game trading app where users can trade unused game keys. This lets players keep an inventory of game keys and exchange the ones they don't want for new games. It was made as a final project for a coding bootcamp. Currently offline.

GitHub link: Git Repository
Application link: UpTrade

GLOBE PLOTTER

Globe Plotter is a full-stack web application developed by a team of student programmers that helps the user plan out a vacation to different cities. The app automatically gives five options for each type of places a traveler would want to visit, including restaurants, landmarks and tours. The app allows users to make accounts that will save and track their trips automatically.
