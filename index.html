ABOUT DAN

Coming from a background in literay, journalistic, and legal editing, Dan McKeon is a coder-in-training in both front-end and back-end web development. Dan graduated from the University at Buffalo in 2018, spending three years working for The Spectrum, the student-run newspaper, as a reporter, copy chief and managing editor, as well as three years as an editor for NAME Magazine, the literary magazine.

Out of college, Dan worked for General Code editing municipal ordinances into code books available online in Rochester, NY. Dan currently works in the Philadelphia suburbs as a barista for Starbucks and as a communications coordinator for the Church of the Holy Apostles while learning full-stack web development through the coding bootcamp at UPenn.

Dan also enjoy reading and writing poetry, playing video games, and scripting Twitter bots to tell jokes.


PORTFOLIO

DOOSKBOT

Dooskbot is a Twitter bot that runs on JavaScript. It uses a bank of nouns, verbs, adjectives and other parts of speech to create tweets based on a collection of templates. The bot uses randomly generated numbers to pick both the type of tweet and the specific words it will use in the tweet, and then automatically sends it out to the World.

GitHub link: Git Repository
Twitter link: @dooskbot

SPACE!

Space! is a web application developed by a team of students for a bootcamp project. It asks for the user's name on first use and then pulls various space stats from several APIs, including multiple NASA ones, to give a list of quick hits about the cosmos, including how many humans are in space and a Martian weather report.

GitHub link: Git Repository
Application link: Space!

WEATHER DASHBOARD

Weather Dashboard is, as the name would suggest, a dashboard of weather! Users can search a city and get a detailed current weather report, including metrics such as current temperature, wind speed, and UV index. The dashboard also generates a five-day forecast for the chosen city and saves recent searches for quick reference.

GitHub link: Git Repository
Application link: Weather Dashboard
