ABOUT DAN

Coming from a background in literay, journalistic, and legal editing, Dan McKeon is an LGBTQ coder-in-training in both front-end and back-end web development. Dan graduated from the University at Buffalo in 2018, spending three years working for The Spectrum, the student-run newspaper, as a reporter, copy chief and managing editor, as well as three years as an editor for NAME Magazine, the literary magazine.

Out of college, Dan worked for General Code editing municipal ordinances into code books available online in Rochester, NY. They currently work in the Philadelphia suburbs as a barista for Starbucks and as a communications coordinator for the Church of the Holy Apostles while learning full-stack web development through the coding bootcamp at UPenn.

Dan also enjoys reading and writing poetry, playing and modding video games, and programming a Twitter bot to dunk on them.

PORTFOLIO

GLOBE PLOTTER

Globe Plotter is a full-stack web application developed by a team of student programmers that helps the user plan out a vacation to different cities. The app automatically gives five options for each type of places a traveler would want to visit, including restaurants, landmarks and tours. The app allows users to make accounts that will save and track their trips automatically.

GitHub link: Git Repository
Application link: Globe Plotter

SPACE!

Space! is a web application developed by a team of student programmers. It asks for the user's name on first use and then pulls various space stats from several APIs, including multiple NASA ones, to give a list of quick hits about the cosmos, including how many humans are in space and a Martian weather report.

GitHub link: Git Repository
Application link: Space!

DOOSKBOT

Dooskbot is a Twitter bot that runs on Node.js made as a side project to try out concepts learned in class. It uses a bank of nouns, verbs, adjectives and other parts of speech to create tweets based on a collection of templates. The bot uses RNG to pick both the type of tweet and the specific words it will use in the tweet, as well as replies, and then automatically sends it out to the World.

GitHub link: Git Repository
Twitter link: @dooskbot

WEATHER DASHBOARD

Weather Dashboard is, as the name would suggest, a dashboard of weather! Users can search a city and get a detailed current weather report, including metrics such as current temperature, wind speed, and UV index. The dashboard also generates a five-day forecast for the chosen city and saves recent searches for quick reference.

GitHub link: Git Repository
Application link: Weather Dashboard

SQUAD UP

Squad Up is a Node.js application that goes through a series of questions to help the user assemble a simple HTML directory for their team. The app uses jest for testing purposes as well as inquirer to ask and record the answers to a series of prompts, and then generates an HTML page using those recorded prompts.

GitHub link: Git Repository

NOTE TAKER

Note Taker is an app that simply takes and saves any notes the user wants and displays them in a clear-cut list for quick reference. The app takes a title and a body for each note and saves them on a server-side database.

GitHub link: Git Repository
Application link: Note Taker
